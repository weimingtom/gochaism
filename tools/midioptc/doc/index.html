midioptc

概要

標準MIDIファイル（SMF）から不要なメタイベントを削除します。Cherryの出力に必要ないデータがたくさんくっついてくるので書きました。おそらく需要もなければ、アップデートする気もないソフトウェアです。

使用方法

悟れ、と言いたいところですが、過去のドキュメントの内容を置いておきます。

プログラムを起動させ、入力ボックスに変換対象ファイルのパスを入力してください。「参照」ボタンでダイアログから選択したり、ウィンドウに対象ファイルをドロップしたりしても、この入力ボックスにファイルパスを反映できます。また、ワイルドカード（例：*.mid）を指定することで複数ファイルの処理が一度で実行可能です。

ファイルパスを入力したら「変換」ボタンを押せば変換処理が開始されます。変換の詳細設定は、「設定」ボタンを押して表示されるダイアログでおこなってください。設定ダイアログから行える設定項目の詳細は以下のとおりです。

変換終了時にダイアログを表示
すべての変換が終了した際に変換終了を示すダイアログを表示します。このチェックがされていない場合には代わりにビープ音を再生します。
変換時にバックアップ(*.bak)を作成する
変換開始の際にバックアップファイルを作成します。変換の挙動にちょっとでも不安を感じたらチェックすることオススメ。
マーカーのメタイベントを削除する
Cherryでは拍子、調号、マーカーのメタイベント操作を一括で行います。ゆえにマーカー書き込みの必要がなくても書き込まれます。その除去に。
調号のメタイベントを削除する
Cherryでは拍子／調号／マーカーのメタイベント操作を一括で行います。ゆえに調号書き込みの必要がなくても書き込まれます。その除去に。
テキストと歌詞のメタイベントを削除
テキストと歌詞のイベントをすべて削除します。別に Cherry でも削除できますが、なんとなく実装。
曲名とトラック名と著作権のテキストを削除
これらテキスト系のイベントを一気に抹消します。
トラック名のテキストを削除
トラック名のテキストを削除します。
フォーマット 1 のデータをフォーマット 0 に変換する（未実装）
Cherryで生成できるデータはフォーマット1のみです。フォーマット0が作成できなくて困っている方はどうぞ。
